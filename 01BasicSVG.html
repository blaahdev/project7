3. VIEWPORT & VIEWBOX

this is the VIEWPORT. its a window to the svg canvas. svg is infinite. hence this viewport allows to "view" which part of the svg canvas you wanna view. it will explicitly add viewbox="0 0 yourWidth yourHeight"
viewBox="min-x min-y width height"
min-x and min-y specify the starting point of the viewBox within the SVGâ€™s coordinate system. The width and height within the viewBox control the visible area and scaling of the content within that coordinate system, regardless of the SVG element's width and height.

you can also explicitly state the viewbox

if your svg width and height does not match the viewbox, it still works. the svg width and height is like the x and y axis counting grid in px, while the viewbox-width and viewbox-height is the viewing width. Hence, allow the view to zoom in or out.
